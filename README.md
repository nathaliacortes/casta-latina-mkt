# Casta Latina — Plataforma de Marketing

## Deploy en Vercel (5 minutos)

1. Sube esta carpeta a un repositorio en GitHub
2. Ve a vercel.com → "New Project" → importa el repo
3. Vercel detecta Vite automáticamente → clic en "Deploy"
4. En ~1 minuto tienes tu URL pública

## Desarrollo local

```bash
npm install
npm run dev
```

## Base de datos
Usa Supabase. Las credenciales están en `src/App.jsx` (líneas 4-5).
